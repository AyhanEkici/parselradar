# Admin Evidence Review and Ranking Policy

## Admin Review Flow
1. Check source type.
2. Check whether parcel identity is clear.
3. Check whether price and m2 are documented.
4. Check whether ada/parsel exists.
5. Check whether zoning claim is supported.
6. Check whether UCBP/TUCBS/e-Devlet evidence is informational only.
7. Check whether sensitive data is redacted.
8. Assign evidence confidence.
9. Assign opportunity status.
10. Decide whether ready for report or AI ranking.

## Required Policy Constraints
- no official legal/tapu/imar guarantee
- manual evidence does not equal official connector
- AI ranking only after source classification
- admin can override confidence with reason
- every high-potential opportunity must show missing verification steps

## Override Governance
- Any confidence override must include reviewer name, timestamp, and reason.
- Overrides cannot remove source-classification provenance.
- Overrides cannot elevate unverified evidence to approved-connector certainty.
