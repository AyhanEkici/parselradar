## 9) Red-Flag Answers
- Viewer thinks the report is official verification.
- Viewer interprets output as official valuation.
- Viewer believes TKGM/e-Imar is automated now.
- Viewer perceives buy/sell advice claim.
- Viewer cannot identify practical usage context.

## 10) What Counts As Useful Feedback
- Specific confusion point with route/screen reference.
- Specific value moment tied to workflow step.
- Clear trust reaction to disclaimers/boundaries.
- Concrete improvement request with reason.

## 11) What Counts As Low-Signal Feedback
- "Everything is good" without examples.
- Generic praise with no route-specific detail.
- No concrete confusion, value, or trust statements.

## 12) How To Record Answers
- Capture each required question verbatim.
