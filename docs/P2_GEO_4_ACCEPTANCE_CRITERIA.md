# P2_GEO_4 Acceptance Criteria

## P2.GEO-5 Start Gate
P2.GEO-5 can start only when all are true:
- Source/license plan documented.
- Import script design approved.
- PostGIS schema stable.
- Kayseri signal test remains PASS.
- Basic Risk Scan remains green.
- No dirty unrelated workspace in staged scope.
- No secrets staged.
- No connector activation.
- No scraping.
- No official verification claim.

## Future Implementation Proof Requirements
Future implementation must prove:
- Turkey extract downloaded.
- Import completes successfully.
- Indexes built successfully.
- Signal queries pass expected contract checks.
- Attribution is present in user-visible/report output.
- Fallback behavior works when no nearby features are found.
- Performance baseline is recorded and reviewed.

## Out Of Scope For This Phase
- Full Turkey import execution.
- Scheduled update automation.
- Connector activation.
- Scraping workflows.
